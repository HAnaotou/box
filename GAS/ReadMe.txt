outputSheet.gsについて

入力として，以下のようなデータを受け取ります．

データ形式{
  state: 処理内容(登録，削除，出席，出席記録保存，業務終了)
  name: 名前
  freeword(?): フリーワード
  mail1: メールアドレス1
  ...
  mail5: メールアドレス5
  time: 時刻
 }


stateについて
入力として受け取るstateは以下の動作を起こします．
state="register":
	生徒情報を新規に登録します．
	出席管理シート，記録先一覧シートに新たに名前を追加し，生徒の出席記録保存用のスプレッドシートを新たに作成します．

state="delete":
	生徒情報を削除します．
	出席管理シート，記録先一覧シートから名前を削除します．(出席記録管理シートは残ります．)

state="attend":
	リアルタイムの生徒の出席状況を更新します．
	出席管理シートの出席状況を更新します．

state="update":
	出席管理シートの情報を，生徒全員の出席記録に反映させます．

state="reset":
	出席管理シートの情報を更新し，リアルタイム出席状況をリセットします．


name，freewordについて
freewordの有無によって，以下の様に表示名を決定します．

freeword有: name(freeword)
freeword無: name


関数について
Register():
	出席管理シート，記録先一覧シートへ登録名を追加し，出席が反映される状態にします．
	さらに出席記録を保存するためのシートを作成します．
	すでに同じ名前が登録されている場合はエラーを返します．

Delete():
	出席管理シート，記録先一覧シートに登録されている名前を削除し，出席管理が反映されない状態にします．
	出席記録を保存するためのシートは削除されません．
	出席管理シート，記録先一覧シートに名前がない場合はエラーを返します．

Attend():
	出席管理シートに出席の状態，出席時刻，退席時刻を記入します．
	出席管理シートに名前がない場合はエラーを返します．

Update():
	出席管理シートの情報を，すべての生徒の出席記録シートに反映します．
	出席管理シートにある名前に対応する，出席記録シートがない場合はエラーを返します．

Reset():
	出席管理シートの出席状況，出席時刻，退席時刻を初期化します．
